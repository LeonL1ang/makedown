# Vue操作手册





### vue-ci初始化项目



##### 安装vue-cli脚手架：

```
cnpm install vue-cli -g       #全局安装vue-cli
```



##### 初始化一个项目：

```
vue init webpack vue_demo      #用webpack模板创建一个项目
```







### 引入element-ui



1. 打开cmd，在项目的根目录下输入：

   ``` 
   npm install element-ui -S  # -S：安装到上线环境 -D：安装到开发环境
   ```
